Detailed Description

template<class string_type_ = hpx::util::logging::hold_string_type, class ptr_type = void*>
struct hpx::util::logging::optimize::cache_string_several_str< string_type_, ptr_type >

This holds 3 strings - one for prepend, one for modification, and one for appending.

When you prepend or append, you can also specify an extra argument

  • an identifier. This identifier uniquely identifies the prepended or appended message.

Afterwards, you can prepend/append only by specifying an identifier

  • which will identify a previously appended or prepended message

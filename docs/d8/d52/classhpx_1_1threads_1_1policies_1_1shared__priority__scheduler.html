Detailed Description

template<typename Mutex = compat::mutex, typename PendingQueuing = lockfree_abp_fifo, typename StagedQueuing = lockfree_fifo, typename TerminatedQueuing = lockfree_lifo>
class hpx::threads::policies::shared_priority_scheduler< Mutex, PendingQueuing, StagedQueuing, TerminatedQueuing >

The shared_priority_scheduler maintains exactly one queue of work items (threads) per OS thread, where this OS thread pulls its next work from. Additionally it maintains separate queues: several for high priority threads and one for low priority threads. High priority threads are executed by the first N OS threads before any other work is executed. Low priority threads are executed by the last OS thread whenever no other work is available.

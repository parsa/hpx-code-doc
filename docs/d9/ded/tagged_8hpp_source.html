196 // A tagged getter is an empty trivial class type that has a named member
197 // function that returns a reference to a member of a tuple-like object that is
198 // assumed to be derived from the getter class. The tuple-like type of a tagged
199 // getter is called its DerivedCharacteristic. The index of the tuple element
200 // returned from the getter's member functions is called its ElementIndex. The
201 // name of the getter's member function is called its ElementName.
202 
203 // A tagged getter class with DerivedCharacteristic D, ElementIndex N, and
204 // ElementName name shall provide the following interface:
205 //
206 // struct __TAGGED_GETTER
207 // {
208 // constexpr decltype(auto)
209 // name() & { return get<N>(static_cast<D&>(*this)); }
210 // constexpr decltype(auto)
211 // name() && { return get<N>(static_cast<D&&>(*this)); }
212 // constexpr decltype(auto)
213 // name() const & { return get<N>(static_cast<const D&>(*this)); }
214 // };

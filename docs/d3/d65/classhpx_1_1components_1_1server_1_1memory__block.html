inlinestatic

The memory for managed_component objects is managed by a class specific allocator. This allocator uses a one size heap implementation, ensuring fast memory allocation. Additionally the heap registers the allocated managed_component instance with the AGAS service.

Parameters
size[in] The parameter size is supplied by the compiler and contains the number of bytes to allocate.

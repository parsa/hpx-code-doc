◆ reserve()

template<typename T , typename Data >
HPX_PARTITIONED_VECTOR_SPECIALIZATION_EXPORT void hpx::server::partitioned_vector< T, Data >::reserve ( size_type  n)

Request the change in partitioned_vector_partition capacity so that it can hold n elements.

This function request partitioned_vector_partition capacity should be at least enough to contain n elements. If n is greater than current partitioned_vector_partition capacity, the function causes the partitioned_vector_partition to reallocate its storage increasing its capacity to n (or greater). In other cases the partitioned_vector_partition capacity does not got affected. It does not change the partitioned_vector_partition size.

Parameters
nminimum capacity of partitioned_vector_partition

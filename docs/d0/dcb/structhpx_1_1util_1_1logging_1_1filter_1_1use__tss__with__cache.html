Detailed Description

template<int default_cache_secs = 5>
struct hpx::util::logging::filter::use_tss_with_cache< default_cache_secs >

Uses TSS (Thread Specific Storage) to find out if a filter is enabled or not.

It caches the current "is_enabled" on each thread. Then, at a given period, it retrieves the real "is_enabled".

Remarks

Another implementation can be done, which could be faster

  • where you retrieve the "is_enabled" each X calls on a given thread (like, every 20 calls on a given thread)

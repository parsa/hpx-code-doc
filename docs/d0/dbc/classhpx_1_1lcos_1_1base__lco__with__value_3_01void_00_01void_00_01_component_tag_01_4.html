Detailed Description

template<typename ComponentTag>
class hpx::lcos::base_lco_with_value< void, void, ComponentTag >

The base_lco<void> specialization is used whenever the set_event action for a particular LCO doesn't carry any argument.

Template Parameters
voidThis specialization expects no result value and is almost completely equivalent to the plain base_lco.

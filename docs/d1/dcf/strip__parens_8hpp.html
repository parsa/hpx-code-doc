◆ HPX_PP_STRIP_PARENS

#define HPX_PP_STRIP_PARENS (   X)    HPX_PP_DETAILS_EVAL((HPX_PP_DETAILS_STRIP_PARENS_I X), X) \

For any symbol X, this macro returns the same symbol from which potential outer parens have been removed. If no outer parens are found, this macros evaluates to X itself without error.

The original implementation of this macro is from Steven Watanbe as shown in http://article.gmane.org/gmane.comp.lib.boost.user/61011

Detailed Description

template<std::size_t Depth>
struct hpx::util::detail::future_unwrap_until_depth< Depth >

A mapper that maps futures to its representing type

The mapper does unwrap futures nested inside futures until the particular given depth.

  • Depth > 1 -> Depth remaining
  • Depth == 1 -> One depth remaining
  • Depth == 0 -> Unlimited depths

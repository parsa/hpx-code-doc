Defaults, and overriding them.

The logging lib has a few default types, used throughout the lib. They are:

  • char_type - the char type used throught the lib; by default, it's char
  • hold_string_type - the type used to hold a string; by default, it's std::string
  • filter_type - the default filter; by default, it's filter::no_ts
  • lock_resource - used to lock resources for access. See locker namespace.
  • mutex - the mutex class used throughout the library. By default,
  • it's mutex_win32 for Windows, or mutex_posix for POSIX

They are all present in default_types structure.

If you want to override any of the above, you should do the following:

  • before including anything from Boost Logging Library, #include <hpx/util/logging/defaults.hpp>
  • override the types
  • do #include <hpx/util/logging/logging.hpp>

Example:

#include <hpx/util/logging/defaults.hpp>
namespace hpx { namespace util { namespace logging {
struct types<override> : default_types {
// define your types
typedef wchar_t char_type;
// etc.
};
}}}
#include <hpx/util/logging/logging.hpp>

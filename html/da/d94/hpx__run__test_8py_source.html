10 # hpx_run_test.py is a script that is designed to manage the execution of HPX
11 # tests. It is designed to:
12 #
13 # * Robustly clean up failed tests by killing them and all their subprocesses.
14 # * Robustly enforce user-specified timeouts for tests with minimal overheads.
15 # * Control the output of tests.
16 #
17 # hpx_run_test.py takes a number of command line arguments, described in
18 # --help. Additionally, it takes a positional argument that describes
19 # the test. This argument should be formatted as a python list. Its format is
20 # described in --help.

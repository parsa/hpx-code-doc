559  // - The first only takes a callable object which accepts a future
560  // object as a parameter.
561  // - The second function takes an executor as the first parameter
562  // and a callable object as the second parameter.
563  // - The third function takes a launch policy as the first parameter
564  // and a callable object as the second parameter.
565  // In cases where 'decltype(func(*this))' is future<R>, the
566  // resulting type is future<R> instead of future<future<R>>.
567  // Effects:
568  // - The continuation is called when the object's shared state is
569  // ready (has a value or exception stored).
570  // - The continuation launches according to the specified launch
571  // policy or executor.
572  // - When the executor or launch policy is not provided the
573  // continuation inherits the parent's launch policy or executor.
574  // - If the parent was created with std::promise or with a
575  // packaged_task (has no associated launch policy), the
576  // continuation behaves the same as the third overload with a

6 This directory contains python script to help debugging HPX applications with
7 gdb. In order to use it, please add the following lines to your ~/.gdbinit:
8 
9  source $HPX_SRC_DIR/tools/gdb/hpx.py
10 
11  define hook-continue
12  hpx thread restore
13  end
14 
15 For a list of commands see `help hpx` inside of gdb.
16 
17 Notes:
18 
19  - The scripts currently only work when HPX is compiled against Boost 1.56
20  - The scripts currently only work with the local priority scheduler
21  - The scripts currently only work on 64 bit executables
22  - The scripts currently only work with the `x86_linux_context` context
23  implementation (which is the default on Linux X86_64)
24  - The hook-continue is needed because we currently just overwrite the current
25  frame with the selected HPX user level context.
26 
27 By not doing a `hpx thread restore` before gdb continues execution, your
28 program will abort.

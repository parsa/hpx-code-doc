Detailed Description

template<typename Connection, typename Key>
class hpx::util::connection_cache< Connection, Key >

This class implements an LRU cache to hold connections. It includes entries checked out from the cache in its cache size.

◆ get_or_reserve()

template<typename Connection, typename Key>
bool hpx::util::connection_cache< Connection, Key >::get_or_reserve ( key_type const &  l,
connection_type &  conn,
bool  force_insert = false 
)
inline

Try to get a connection to l from the cache, or reserve space for a new connection to l. This function may evict entries from the cache.

Returns
If a connection was found in the cache, its value is assigned to conn and this function returns true. If a connection was not found but space was reserved, conn is set such that conn.get() == 0, and this function returns true. If a connection could not be found and space could not be returned, conn is unmodified and this function returns false. If force_insert is true, a new connection entry will be created even if that means the cache limits will be exceeded.
Note
The connection must be returned to the cache by calling reclaim().

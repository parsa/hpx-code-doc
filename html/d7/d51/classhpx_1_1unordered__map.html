Detailed Description

template<typename Key, typename T, typename Hash, typename KeyEqual>
class hpx::unordered_map< Key, T, Hash, KeyEqual >

This is the unordered_map class which defines hpx::unordered_map functionality.

This contains the client side implementation of the hpx::unordered_map. This class defines the synchronous and asynchronous API's for each of the exposed functionalities.
